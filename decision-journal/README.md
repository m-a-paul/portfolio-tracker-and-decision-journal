# Decision Journal

Use one Markdown file per meaningful trade idea, whether entered or skipped.

Suggested naming:

```text
YYYY-MM-DD-ticker-short-description.md
```

Examples:

```text
2026-06-10-crnc-call-add.md
2026-06-10-pltr-trim-plan.md
2026-06-10-cpi-risk-plan.md
```

The journal is for the thinking: thesis, chart, risk, emotions, and lessons. The spreadsheet is for the numbers: positions, premiums, realized P/L, weights, and cash.
